class的命名:

(1)顏色:使用顏色的名稱或者16進制代碼,如
.red { color: red; }
.f60 { color: #f60; }
.ff8600 { color: #ff8600; }

(2)字體大小,直接使用"font+字體大小"作為名稱,如
.font12px { font-size: 12px; }
.font9pt {font-size: 9pt; }

(3)對齊樣式,使用對齊目標的英文名稱,如
.left { float:left; }
.bottom { float:bottom; }

(4)標題欄樣式,使用"類別+功能"的方式命名,如
.barnews { }
.barproduct { }

* 注意事項:

  1. 一律小寫。
  2. 盡量用英文。
  3. 盡量不縮寫，除非一看就明白的單詞。
